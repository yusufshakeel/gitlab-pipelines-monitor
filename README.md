## Prerequisite

- Node (v14 or higher)

## Getting started

Install the package globally.

```shell
npm i -g glpm
```

## Run

```shell
glpm
```

## Initialise

```shell
glpm init
```

## Check status

```shell
glpm status
```

Run in watch mode.

```shell
glpm status -watch -interval=10000
```

## Help

```shell
glpm --help
```
